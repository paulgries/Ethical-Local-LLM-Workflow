# Ethical-Local-LLM-Workflow
A journey: learning agentic LLM workflows

My goal is to experiment with local development with an eye toward both the environment and ethics. I own an M1 Max MacBook Pro with 32GB memory.

I chose StarCoder2-15B-inspect, which is trained on license-filtered open source code (The Stack v2), with documented exclusions and opt out mechanisms. In addition to having far fewer ethical issues than the major cloud LLMs, using it locally avoids many IP risks.

I was inspired to do this project by a [LinkedIn post by David Jorjani](https://www.linkedin.com/posts/jorjani_a-few-weeks-ago-i-taught-a-bonus-lecture-share-7452379429445668865-V6nK) that chronicled his 50-minute lecture on AI where he designed, built, tested, and deployed 3 new real, live product features using a series of AI agents.
